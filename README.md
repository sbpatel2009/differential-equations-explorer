# Differential Equations Explorer

A web-based interactive tool to:

- **Enter an equation or system of differential equations**
- **Visualize numerical solutions**
- **Draw phase portraits**

## Features

- Equation input widgets
- Plots of numerical solutions
- Phase portrait generator

## Getting Started

1. Clone this repo.
2. Open `index.html` in your browser.
3. Enter differential equations, initial conditions, and view results interactively!

## Technologies

- JavaScript
- [math.js](https://mathjs.org/)
- [Plotly.js](https://plotly.com/javascript/)
